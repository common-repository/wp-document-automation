== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'.
2. Search for `WP Document Automation`.
3. Activate `WP Document Automation` integration from your Plugins page.

= From WordPress.org =

1. Visit (https://wordpress.org/plugins/wp-document-automation/)[https://wordpress.org/plugins/wp-document-automation/] and download plugin.
2. Upload the `wp-document-automation` directory to your '/wp-content/plugins/' directory.
3. Activate `WordPress Document Automation` plugin from your Plugins page.

OR

1. Visit your WordPress Admin > Plugins > Add Plugin
2. Search for 'WordPress Document Automation'
3. Install the plugin created by wpt00ls and activate.
